## 💻 开发环境

### 必需工具
- **Qt Creator** - 官方 IDE
- **Qt 6.5+** - Qt 框架
- **CMake** - 构建工具

### 推荐工具
- **Git** - 版本控制
- **VS Code** - 代码编辑器 (可选)
- **Qt Design Studio** - 可视化设计工具

### 安装步骤

1. **下载 Qt**
   - 访问 [Qt 官网](https://www.qt.io/download)
   - 下载 Qt Online Installer
   - 安装 Qt 6.5 或更高版本

2. **配置 Qt Creator**
   - 打开 Qt Creator
   - 配置 Kit (编译器、Qt 版本、CMake)
   - 创建第一个项目

3. **验证安装**
   ```qml
   import QtQuick 6.5
   import QtQuick.Controls 6.5
   
   ApplicationWindow {
       width: 400
       height: 300
       visible: true
       title: "Hello Qt"
       
       Text {
           text: "Qt 安装成功!"
           anchors.centerIn: parent
           font.pixelSize: 24
       }
   }
   ```

## ❓ 常见问题

### Q: 需要什么基础?
A: 基本的编程概念即可,不需要 C++ 基础。有 JavaScript 经验更好。

### Q: 学习周期多长?
A: 
- 基础入门: 1-2 周
- 熟练掌握: 1-2 个月
- 精通应用: 3-6 个月

### Q: QML 和 Qt Widgets 的区别?
A: 
- QML: 声明式,适合现代UI,性能好
- Qt Widgets: 命令式,传统桌面应用,稳定成熟

### Q: 如何调试 QML?
A: 
- 使用 console.log() 输出日志
- Qt Creator 的调试器
- QML Profiler 性能分析

### Q: 如何与 C++ 交互?
A: 
- 通过 Q_PROPERTY 暴露属性
- 通过 Q_INVOKABLE 暴露方法
- 使用 QML 上下文属性
