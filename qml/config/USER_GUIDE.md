# 动态表单配置编辑器 - 用户操作指南

## 🎯 控件操作指南

### 添加控件
1. 在"添加控件"区域点击相应的控件按钮
2. 控件会自动添加到网格的第一个空位置
3. 添加后的控件会在"控件预览"区域显示

### 配置控件 🔧
**操作方式**: 左键点击已添加的控件
**功能**: 
- 修改控件标识、标签、位置
- 设置控件特定属性（占位符、选项等）
- 配置事件处理代码
- 调整控件在网格中的跨度

### 删除控件 🗑️
**操作方式**: 右键点击要删除的控件
**功能**: 
- 立即从配置中移除控件
- 释放网格位置供其他控件使用
- 自动更新预览显示

## 🎨 视觉提示

### 控件状态指示
- **空单元格**: 白色背景，显示"Cell X (row,col)"
- **有控件单元格**: 彩色背景，显示控件图标和标签
- **鼠标悬停**: 背景变亮，边框加粗
- **操作提示**: 悬停时显示"左键配置 右键删除"

### 控件类型颜色
- 📝 **文本框**: 浅蓝色 (#e3f2fd)
- 🔢 **数字框**: 浅绿色 (#e8f5e8)  
- 🔒 **密码框**: 浅紫色 (#f3e5f5)
- 📋 **下拉框**: 浅橙色 (#fff3e0)
- ☑️ **复选框**: 浅红色 (#ffebee)
- 🔘 **单选框**: 浅灰色 (#f5f5f5)
- 🎯 **按钮**: 浅红色 (#ffebee)

## ⚙️ 网格配置

### 基础设置
- **行数/列数**: 使用SpinBox调整网格尺寸
- **间距**: 设置行间距和列间距（像素）

### 高级设置
- **行高比例**: 逗号分隔的数字，如"1,1,2,1"表示第3行是其他行的2倍高
- **列宽比例**: 逗号分隔的数字，如"1,2"表示第2列是第1列的2倍宽

### 实时预览
- 所有配置变更都会立即在网格预览中生效
- 网格会根据内容自动调整高度
- 支持最多20行10列的网格布局

## 💾 配置管理

### 应用配置
- 点击"应用配置"将当前配置应用到表单预览
- 可以在"表单预览"标签页查看实际效果

### 导出配置
- 点击"导出配置"将配置输出到控制台
- 可以复制JSON配置用于其他用途

### 重置配置
- 点击"重置配置"恢复到默认的8x2网格
- 会清空所有已添加的控件

## 🐛 故障排除

### 控件没有显示
- 检查控制台是否有错误信息
- 确认控件是否添加成功（控制台会显示"添加控件: type"）
- 尝试点击"应用配置"然后查看表单预览

### 网格配置无效
- 检查行高列宽比例格式是否正确（用逗号分隔的数字）
- 确认数值都是正数
- 查看控制台的配置变更日志

### 操作无响应
- 确保点击的是有控件的单元格
- 查看控制台的点击事件日志
- 检查是否有JavaScript错误
