# 快速开始 - 拖拽式表单设计器

## 5分钟创建你的第一个表单

### 步骤 1: 打开拖拽式设计器

1. 启动应用程序
2. 在表单列表页面,点击顶部的 **"拖拽式设计器"** 按钮

![启动设计器](启动设计器.png)

### 步骤 2: 添加控件

#### 方法: 拖拽添加

1. 从左侧控件面板找到 **"📝 文本框"**
2. 按住鼠标左键,拖动到中间画布区域
3. 松开鼠标,控件自动添加

![拖拽添加控件](拖拽添加.png)

**提示**: 重复此操作添加更多控件

### 步骤 3: 编辑控件属性

1. 在画布区域点击刚添加的文本框
2. 控件会高亮显示(蓝色边框)
3. 右侧属性面板自动显示该控件的属性

![选中控件](选中控件.png)

4. 修改以下属性:
   - **控件标识**: 改为 `username`
   - **标签文本**: 改为 `用户名`
   - **占位符**: 改为 `请输入用户名`

5. 点击底部的 **"应用更改"** 按钮

![编辑属性](编辑属性.png)

### 步骤 4: 添加更多控件

按照相同方式添加以下控件:

#### 密码框
- 控件标识: `password`
- 标签文本: `密码`
- 占位符: `请输入密码`

#### 下拉框
- 控件标识: `gender`
- 标签文本: `性别`
- 选项列表:
  ```
  男|male
  女|female
  ```

#### 按钮
- 控件标识: `submit_btn`
- 按钮文本: `提交`

### 步骤 5: 调整控件顺序

如果需要调整控件顺序:
- 点击控件右侧的 **↑** 按钮向上移动
- 点击控件右侧的 **↓** 按钮向下移动

![调整顺序](调整顺序.png)

### 步骤 6: 保存表单

1. 在顶部输入框输入表单名称: `用户登录表单`
2. 点击 **"保存表单"** 按钮
3. 看到成功提示后,自动返回表单列表

![保存表单](保存表单.png)

### 步骤 7: 使用表单

1. 在表单列表中找到刚创建的 `用户登录表单`
2. 点击 **"新增"** 按钮
3. 填写表单并提交

![使用表单](使用表单.png)

## 常见问题

**Q: 如何删除控件?**  
A: 点击控件右侧的 🗑️ 按钮

**Q: 如何修改已保存的表单?**  
A: 在表单列表中点击 "拖拽编辑" 按钮

**Q: 控件标识(key)有什么用?**  
A: 用于唯一标识控件,在数据存储和API调用时使用

**Q: 可以添加事件吗?**  
A: 拖拽式设计器暂不支持事件配置,请使用原配置编辑器
