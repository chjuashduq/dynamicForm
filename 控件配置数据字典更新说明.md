# 控件配置数据字典更新说明

## 更新时间
2025-11-19

## 更新内容

在《动态表单系统完整使用手册.md》中新增了**"控件配置数据字典"**章节，位于"控件类型详解"和"系统架构详解"之间。

## 新增章节概述

### 章节位置
- 目录序号：第5章
- 位置：控件类型详解 → **控件配置数据字典** → 系统架构详解

### 章节内容

新增的数据字典章节提供了类似网格配置面板格式的完整配置参考表，包括：

#### 1. 基本属性配置表
- 所有控件通用的基本属性
- 包含：type, key, label, row, column, rowSpan, colSpan, labelRatio
- 每个属性都标注了：中文名称、数据类型、是否必填、取值范围、默认值、示例

#### 2. 网格布局配置表
- 表单整体布局的配置项
- 包含：rows, columns, rowSpacing, columnSpacing, rowHeights, columnWidths
- 附带完整的JSON配置示例

#### 3. 样式属性配置表
- 控件外观样式的配置项
- 包含：labelColor, labelBold, inputColor, inputFontSize

#### 4. 验证配置表
- 验证函数的配置说明
- 包含可用变量和使用示例

#### 5. 事件配置表
- 各类事件处理的配置项
- 包含：onFocusLost, onTextChanged, onValueChanged, onClicked
- 标注了每个事件适用的控件类型

#### 6. 控件类型特定属性表
为每种控件类型提供专属配置表：
- **文本框（text）**：placeholder, value
- **数字框（number）**：value, minimum, maximum, step
- **密码框（password）**：placeholder
- **下拉框（dropdown）**：options, value
- **复选框（checkbox）**：options, direction
- **单选框（radio）**：options
- **按钮（button）**：text, buttonType

#### 7. 完整配置示例
提供了一个包含所有配置项的完整JSON示例，展示了5个不同类型控件的配置。

#### 8. 配置项快速查询表
- **按功能分类**：位置和布局、外观样式、数据和验证、交互事件、特殊属性
- **按控件类型分类**：列出每种控件的必需属性、可选属性和特有属性

## 设计特点

1. **表格化展示**：采用Markdown表格格式，清晰直观
2. **完整性**：覆盖所有控件类型和配置项
3. **实用性**：每个配置项都包含示例值
4. **分类清晰**：按功能和控件类型双重分类
5. **快速查询**：提供快速查询表，方便开发者快速定位所需配置

## 使用场景

1. **快速查阅**：开发时快速查找某个配置项的含义和用法
2. **配置参考**：编写JSON配置时作为参考手册
3. **新手指南**：帮助新用户快速了解所有可用配置项
4. **API文档**：作为配置API的完整文档

## 与现有章节的关系

- **控件类型详解**：侧重于控件的功能和使用场景说明

- 两者互补，前者是"使用指南"，后者是"技术手册"
